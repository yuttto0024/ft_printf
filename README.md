## Overview

この課題は、C言語の標準関数である `printf` を再実装するプロジェクトである。
フォーマット指定子を解析し、可変長引数を用いて様々なデータ型を標準出力へフォーマット出力する独自の関数 `ft_printf` を作成した。

## Language & Tools

* **Language**: C
* **Build**: Makefile (`make`, `cc`, `ar`)
* **Functions**: `malloc`, `free`, `write`, `va_start`, `va_arg`, `va_copy`, `va_end`

## What I Learned

このプロジェクトを通じ、低レイヤのプログラミングおよびシステム開発の基礎となる以下のスキルを習得した。

* **可変長引数の処理**:  
`stdarg.h` のマクロ（`va_start`, `va_arg`, `va_end`）を利用し、コンパイル時には数や型が定まらない引数を安全に扱う方法を習得した。
* **フォーマット解析と関数ポインタによる分岐制御**:  
文字列を走査してフォーマット指定子（`%c`, `%s`, `%p`, `%d`, `%i`, `%u`, `%x`, `%X`, `%%`）を解析するロジックを実装した。
特に、`switch` 文や大量の `if-else` を避けるため、関数ポインタの配列を活用し、`O(1)` で適切な出力関数を呼び出す設計を行った。
* **システムコールと再帰を用いた出力制御**:  
標準ライブラリのバッファリング機能に依存せず、`write` システムコールのみを用いて文字列や数値を標準出力へ直接書き出す処理を実装した。
また、数値から文字列への変換（10進数、16進数）において再帰呼び出しを行い、コードを簡潔にまとめた。
* **コーディング規約の厳守**:  
42のコーディング規約（Norm）を遵守し、可読性が高く、保守性の高いコードを書く訓練を行った。
